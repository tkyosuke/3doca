# tutorials - チュートリアル（Tier 1）

## 目的

15分以内に最初の成功した操作を可能にする。ミニマリズム原則を適用し、説明するのではなくアクション指向で即座に実行。

## ティア仕様

| 項目 | 値 |
|------|-----|
| **ティア** | Tier 1 |
| **時間目標** | 15分 |
| **ビジュアル/テキスト比率** | 60/40 |
| **対象読者** | 新規ユーザー、学習者 |
| **成功基準** | このティアのみでコアワークフローを完了できる |

## チュートリアルの要件

1. **学習目標を冒頭に明記**
2. **完走可能な手順**（途中で詰まらない）
3. **各ステップに検証ポイント**
4. **所要時間の目安を記載**
5. **スクリーンショット付きの番号付きステップ**
6. **期待される出力を含むコードスニペット**
