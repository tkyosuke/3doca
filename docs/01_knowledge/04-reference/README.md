## 目的

包括的なルックアップドキュメントを提供。線形読書ではなく、検索とスキャンに最適化。

## ティア仕様

| 項目 | 値 |
|------|-----|
| **ティア** | Tier 3 |
| **ビジュアル/テキスト比率** | 20/80 |
| **対象読者** | 仕様を確認したいユーザー |

1. **網羅性を重視**
2. **構造化された形式**（テーブル、定義リスト）
3. **検索しやすいキーワード配置**
4. **バージョン情報必須**
