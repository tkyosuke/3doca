## 目的

目標に焦点を当てた手順ドキュメントを提供。「Xをどのように達成するか？」に答える。

| 項目 | 値 |
|------|-----|
| **ティア** | Tier 2 |
| **ビジュアル/テキスト比率** | 40/60 |
