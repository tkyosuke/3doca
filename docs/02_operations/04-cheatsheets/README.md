# cheatsheets - クイックリファレンス（Tier 0/3）

## 目的

即座参照用の要点集約。1ページ以内で印刷可能なレイアウト。

## ティア仕様

チートシートは用途により2つのティアに分類される：

| タイプ | ティア | ビジュアル/テキスト比率 | 用途 |
|--------|--------|----------------------|------|
| ビジュアル重視 | Tier 0 | 90/10 | 全体像の把握 |
| テキスト重視 | Tier 3 | 20/80 | コマンドリファレンス |

## チートシートの要件

1. **1ページ以内**（Tier 0相当）
2. **視覚的構成**（テーブル、コードブロック中心）
3. **説明は最小限**
4. **印刷可能なレイアウト**
