# playbooks - 状況対応（Tier 2）

## 目的

複雑シナリオの意思決定枠組を提供。判断分岐とエスカレーション基準を明確化。

## ティア仕様

| 項目 | 値 |
|------|-----|
| **ティア** | Tier 2 |
| **ビジュアル/テキスト比率** | 40/60 |
| **必須要素** | トリガー条件、デシジョンツリー |

## プレイブックの要件

1. **トリガー条件を冒頭に**
2. **判断分岐を明確に**（フローチャート推奨）
3. **エスカレーション基準**
4. **完了条件・確認事項**
